#  Simple Login System (Python)

##  Description
This project is a simple login and signup system built using Python. 
It allows users to create an account and log in using a username and password.

The program runs in a step-by-step manner through a command-line interface, 
making it easy to understand and use.

---

##  Features
- User Signup (Create a new account)
- User Login (Verify credentials)
- Menu-driven Command Line Interface

---

##  Tech Used
- Python

2. Run the file:

   python main.py
